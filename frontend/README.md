# Skautu Inventoriaus Valdymas Web Frontend

React + Vite + TypeScript browser client for the independent website backend.

```powershell
cd skautu-inventoriaus-valdymas-web/frontend
npm install
npm run dev
npm run build
```

Set `VITE_API_BASE_URL` to the website backend origin. The local default is `http://localhost:8081`.
